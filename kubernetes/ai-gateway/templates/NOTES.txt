-------------------------------------------------------------------------------
TROUBLESHOOTING
-------------------------------------------------------------------------------

If pods are not starting:

  1. Check Vault connectivity:
     kubectl exec -it <pod> -n {{ .Release.Namespace }} -- cat /vault/secrets/config

  2. Check database connectivity:
     kubectl exec -it <pod> -n {{ .Release.Namespace }} -- nc -zv {{ include "ai-gateway.pgbouncer.serviceName" . }} {{ .Values.pgbouncer.service.port }}

  3. Check pod events:
     kubectl describe pod <pod-name> -n {{ .Release.Namespace }}
